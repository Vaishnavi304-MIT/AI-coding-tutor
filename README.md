# 🤖 AI Coding Tutor Agent

> **An intelligent AI-powered tutor that helps users learn programming, practice problems, and master DSA through interactive, context-aware guidance.**

---

![Demo](https://github.com/user-attachments/assets/e8826f7d-a05a-498e-a8bc-f5b4fdca26f5)

---

## 🚩 Problem Statement

Learning programming is challenging because:

* Beginners struggle with understanding concepts
* Practice platforms don’t provide personalized guidance
* DSA learning lacks step-by-step explanation
* Users get stuck without proper hints or debugging help

👉 Result: **slow learning, frustration, and lack of confidence**

---

## 💡 Solution

An AI Coding Tutor that:

* Teaches programming concepts interactively
* Provides coding practice problems (like LeetCode)
* Guides users step-by-step instead of giving direct answers
* Helps with debugging, optimization, and logic building

---

## 🛠️ Technologies Used

* **Streamlit (Frontend UI)**
* **Python**
* **LangChain**
* **Groq LLM (LLaMA 3.1)**
* **Subprocess (Code Execution Engine)**

---

## ⚙️ Core Features

### 🧠 Learn Mode

* Ask any coding question
* Get clear explanations with examples
* Works like ChatGPT for programming

---

### 🎯 Practice Mode

* Generates coding problems
* Includes:

  * Problem statement
  * Constraints
  * Example
* Provides hints instead of full solutions

---

### 📘 DSA Mode

* Teaches Data Structures & Algorithms
* Includes:

  * Concept explanations
  * Practice problems
  * Logic + optimization

---

### 💻 Code Editor

* Run Python code instantly
* Debug errors
* Optimize code
* Open generated code in editor

---

### ⚡ Smart AI Behavior

* Context-aware responses (Learn / Practice / DSA)
* Intent detection:

  * New problem
  * Hint
  * Solution
  * Explanation
* Guides instead of spoon-feeding

---

## ✨ Wow Factors

* 🔥 Works like a **real coding mentor (not chatbot)**
